# Dashboard SmartApp

Личный дашборд - это SmartApp, который позволяет пользователю закрепить необходимые сервисы, чаты, контакты и открывать SmartApp, чат или карточку контакта прямо из приложения.

В дашборде предусмотрены функции поиска и сортировки сущностей.
